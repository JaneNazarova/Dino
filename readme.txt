Игра 'Динозаврик'.
Игра реализована с помощью библиотеки python pygame.

Правила:
Игра можно как с клавиатуры, так и с мышки.
Левая кнопка мыши, пробел, стелочка вверх, клавиша w - позволяют прыгать.
Правая кнопка мыши, ctrl, стрелочка вниз, клавиша s  - позволяют нагинаться.

Приятной игры!
